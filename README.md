# Description
hrome Extension for Time Tracking and Productivity Analytics is a browser extension that monitors and records the amount of time a user spends on different websites. It helps users understand their browsing habits and improve productivity by showing how much time is spent on productive or distracting sites.

The extension works inside the Google Chrome browser and automatically tracks active tabs and visited websites. It stores usage data and generates simple analytics to help users manage their time more effectively.

The extension is developed using standard web technologies such as HTML, CSS, and JavaScript, along with the Chrome Extension API. It can also use Chrome Storage API to save browsing time data locally.

Key features include:

Tracking time spent on each website

Displaying daily or weekly usage statistics

Identifying productive vs. non-productive websites

Providing insights to improve time management

Running automatically in the background while browsing
